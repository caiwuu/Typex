### 介绍
一款全新3.0架构的富文本编辑器；不依赖contenteditable,不依赖document.execCommand；自主实现选区（Selection）、范围（Range）、光标系统（Caret）、输入系统（Inputor）、事件系统（Action）;并且支持多光标操作。UI和数据分离、原子化操作；支持协同编辑。
### 设计目标
1. 高可扩展性
2. 可维护性
3. 稳定性
4. 易用性
5. 在线协同
6. L2级别
